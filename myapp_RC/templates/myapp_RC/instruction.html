INSTRUCTIONS

1. The reverse coding round will last for 30 minutes.
2. You will be permitted two responses for each question.
3. If the answer provided for 1st response is incorrect, only then will the 2nd response field be enabled.
4. The answers are strictly in the form of integers.
5. Throughout this test, you'll be provided with three lifelines. You can use only a single lifeline for one question. These lifelines are described below.
6. Users are advised not to spam submit wrong answers (Test will not traverse if spam is detected).
7. Users are advised to code their solution in personal or external IDEs to compute the answer.

Marking Scheme:

For 1st response:
+4 marks for correct response.
-0 marks for wrong answer.

For 2nd response:
+2 marks for correct response.
-2 marks for wrong answer.

1

Astro Sub (Lifeline 1)

->This lifeline allows contestants to substitute a seemingly difficult question with an easier one.
->You will be allowed one response only for this lifeline.
->You can only use this lifeline if total Astro-Sub fuel is 3.
->Each correct answer will increase the Astro-Sub fuel by one.
->Once this lifeline is used, the fuel again becomes 0.
->You can use this lifeline more than once.

Marking Scheme:
+4 marks for correct response.
-4 marks for wrong answer.
No second attempt allowed.

2

Time Warp (Lifeline2)

->This lifeline gives you the advantage of having additional time if you answer questions correctly.
->This lifeline can only be used once.
->You only have 30 seconds to activate this lifeline once the question is displayed.
->Marking Scheme for this lifeline remains the same.

Timer Scheme:

For first response :
+5 min for correct response
-2 min for wrong response

For second response :
+3 min for correct response
-3 min for wrong response

3

Galactic Guidance (Lifeline 3)

->This lifeline allows contestants to ask ChatGPT, the virtual assistant, for guidance and advice when unsure about the answer to a question.
->This lifeline can only be used after the contestant has attempted at least 2 questions.
->Marking scheme for this lifeline remains the same.
->This lifeline can only be used once.
